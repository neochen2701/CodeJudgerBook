登入 Code Judger 後，會進入「我的課程」頁面，請選擇要進入的課程。

![我的課程](.gitbook/assets/cjmd03-ke-cheng-00-ke-cheng-shou-ye.png)

## 功能

進入選擇的課程後，共有 3 個功能：

- 課程資訊：課程的學員資訊。
- 練習題組：課程練習的題組題目。
- 考試總管：課程的考試。
- 統計資訊：課程的考試。
